    Analyse

    View large batches of TEI documents in table form to help quickly identify key information. All data is stored in the browser and persists between uses, with no need to create an account to get started.

    Export

    Export your tables in Excel, CSV or JSON formats. The latter can be used to subsequently reload data directly into the table, allowing you to easily share your transformations with others.

    Customise

    Each table layout simply defines the XSLT file to use for the initial transformation and the columns to include. This makes it straightforward to edit the source code and make new layouts available.
